# Method HVAC approved branding

These are the authoritative brand assets for the customer-facing Method HVAC website. Do not generate replacement logos or substitute generic HVAC artwork.

## Asset map

| File | Required use |
| --- | --- |
| `method-hvac-logo.png` | Primary Method HVAC company logo. Use in the site header, mobile navigation, homepage hero, and footer. |
| `method-media-tagline.png` | “IT'S ALL IN THE METHOD” slogan artwork. Use in the homepage hero or a dedicated brand banner. |
| `method-hub-seal.png` | Method Alliance / “Member of the Method” trust seal. The filename is historical; use it as a membership and credibility badge. |
| `method-hvac-seal.png` | Secondary Method HVAC ecosystem / heritage seal. Use sparingly in trust or partner sections. |
| `hvac-bob-powered-by-gemini.svg` | HVAC Bob badge. Use only with the transparent customer quote builder and estimate summary. |

## Presentation rules

- Preserve each asset's aspect ratio.
- Render with transparent backgrounds.
- Use `object-fit: contain`; never crop the artwork.
- Do not recolor, redraw, distort, or replace the supplied artwork.
- Do not place body text over logos or seals.
- Add descriptive alt text.
- Keep the primary logo readable at mobile sizes.

Repository-wide rules are defined in:

- `.github/copilot-instructions.md`
- `src/config/branding.js`
- `src/styles/brand.css`
